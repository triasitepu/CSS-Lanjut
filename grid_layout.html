Kesimpulan skenario 1 adalah program tersebut menggunakan css untuk membuat tata letak grid dengan 3 kolom. Dimana display adalah grid, dan Penggunaan grid-template-columns: auto auto auto; menentukan bahwa grid memiliki 3 kolom dengan lebar otomatis. Kemudian setiap elemen yang memiliki class item_grid mempunyai style tertentu yang telah diatur sesuai class css item_grid.
Kesimpulan skenario 3 adalah program tersebut menambahkan elemen grid-column-gap:100px dan grid-row-gap:50px. Kegunaan dari properti css tersebut adalah mengatur jarak antara kolom-kolom grid menjadi 100px dan jarak antara baris-baris grid menjadi 50px. Ini berarti dalam tata letak grid CSS, akan ada jarak 100px di antara setiap kolom dan 50px di antara setiap baris.
Kesimpulan skenario 4 adalah program tersebut menambahkan properti css grid-gap:50px. Ini digunakan untuk mengatur jarak antara kolom dan baris dalam grid menjadi 50px. Ini berarti akan ada 50px ruang antara setiap kolom dan baris dalam tata letak grid CSS.
